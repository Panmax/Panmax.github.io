иҙҫж”Җзҡ„ Macbook дҪҝз”ЁжҢҮеҢ—

Posted on 2018-02-26 | Comments:

е·Ҙж¬Іе–„е…¶дәӢпјҢеҝ…е…ҲеҲ©е…¶еҷЁ

з®—иө·жқҘжҲ‘з”Ё Macbook д№ҹжңүдёүе№ҙеӨҡзҡ„ж—¶й—ҙдәҶпјҢдёӯй—ҙз”ұдәҺдёҖж¬Ўе·ҘдҪңеҺҹеӣ пјҢејҖеҸ‘зҺҜеўғдёҖзӣҙжңүй—®йўҳпјҢе°ұйҮҚиЈ…дәҶдёҖж¬Ўзі»з»ҹпјҢд№ҹд»…д»…иҝҷдёҖж¬ЎпјҢиҖҢдё”иҝҷж¬Ўе…¶е®һд№ҹжҳҜеҶӨжһүдәҶзі»з»ҹпјҢеҪ“ж—¶жҳҜеӣ дёәжҲ‘зҡ„ host ж–Үд»¶жІЎжңүй…ҚзҪ®жӯЈзЎ®еҜјиҮҙзҡ„пјҢеҸҜд»ҘиҜҙ macOS жҳҜзӣёеҪ“зЁіе®ҡзҡ„пјҢе№¶дё”д№ҹжҳҜејҖеҸ‘еҲ©еҷЁпјҢжҲ‘еҶҷиҝҷзҜҮж–Үз« зҡ„зӣ®зҡ„дё»иҰҒжҳҜдёәдәҶд№ӢеҗҺиҮӘе·ұеңЁжҚўж–°зҡ„ Macbook ж—¶жңүи®°еҪ•еҸҜеҜ»пјҢеҗҢж—¶её®еҠ©е…¶д»– Macbook з”ЁжҲ·жқҘеҸ‘зҺ°дёҖдәӣеҘҪз”Ёзҡ„е·Ҙе…·гҖӮ

д»ҘдёӢзҘһеҷЁжҺ’еҗҚеҲҶе…ҲеҗҺ

iTerm2пјҡз»Ҳз«ҜзҘһеҷЁ

iTerm2 жҳҜ Mac дёӢжңҖеҘҪзҡ„з»Ҳз«Ҝе·Ҙе…·пјҢеӨ§йғЁеҲҶеҠҹиғҪйғҪжҳҜејҖз®ұеҚіз”ЁпјҢз®ҖеҚ•д»Ӣз»ҚдёӢ iTerm2 зҡ„зү№иүІеҠҹиғҪпјҡ

жҷәиғҪйҖүдёӯ

еңЁ iTerm2 дёӯпјҢеҸҢеҮ»йҖүдёӯеҚ•иҜҚпјҢдёүеҮ»йҖүдёӯж•ҙиЎҢпјҢеӣӣеҮ»жҷәиғҪйҖүдёӯпјҲжҷәиғҪи§„еҲҷеҸҜй…ҚзҪ®пјүпјҢеҸҜд»ҘиҜҶеҲ«зҪ‘еқҖпјҢеј•еҸ·еј•иө·зҡ„еӯ—з¬ҰдёІпјҢйӮ®з®ұең°еқҖзӯүгҖӮпјҲеҫҲеӨҡж—¶еҖҷеҸҢеҮ»зҡ„йҖүдёӯе°ұе·Із»ҸеҫҲжҷәиғҪдәҶпјү

еңЁ iTerm2 дёӯпјҢйҖүдёӯеҚіеӨҚеҲ¶гҖӮеҚід»»дҪ•йҖүдёӯзҠ¶жҖҒзҡ„еӯ—з¬ҰдёІйғҪиў«ж”ҫеҲ°дәҶзі»з»ҹеүӘеҲҮжқҝдёӯгҖӮ

е·§з”Ё Command й”®

жҢүдҪҸвҢҳй”®пјҡ

  • еҸҜд»ҘжӢ–жӢҪйҖүдёӯзҡ„еӯ—з¬ҰдёІ
  • зӮ№еҮ» URLпјҡи°ғз”Ёй»ҳи®ӨжөҸи§ҲеҷЁи®ҝй—®иҜҘзҪ‘еқҖ
  • зӮ№еҮ»ж–Үд»¶пјҡи°ғз”Ёй»ҳи®ӨзЁӢеәҸжү“ејҖж–Үд»¶
  • зӮ№еҮ»ж–Үд»¶еӨ№пјҡеңЁ finder дёӯжү“ејҖиҜҘж–Үд»¶еӨ№
  • еҗҢж—¶жҢүдҪҸ Option й”®пјҢеҸҜд»Ҙд»Ҙзҹ©еҪўйҖүдёӯ

еёёз”Ёеҝ«жҚ·й”®

  • еҲҮжҚў tabпјҡвҢҳ+вҶҗ, вҢҳ+вҶ’, вҢҳ+{, вҢҳ+}гҖӮвҢҳ+ж•°еӯ— зӣҙжҺҘе®ҡдҪҚеҲ°иҜҘ tab
  • ж–°е»ә tabпјҡвҢҳ+t
  • йЎәеәҸеҲҮжҚў paneпјҡвҢҳ+[, вҢҳ+]
  • жҢүж–№еҗ‘еҲҮжҚў paneпјҡвҢҳ+Option+ж–№еҗ‘й”®
  • еҲҮеҲҶеұҸе№•пјҡвҢҳ+d ж°ҙе№іеҲҮеҲҶпјҢвҢҳ+Shift+d еһӮзӣҙеҲҮеҲҶ
  • жҷәиғҪжҹҘжүҫпјҢж”ҜжҢҒжӯЈеҲҷжҹҘжүҫпјҡвҢҳ+fгҖӮ

з”ЁдәҺжҗңзҙўе…ій”®еӯ—пјҢжҢү Tab й”®еҸҜд»ҘиҮӘеҠЁиЎҘе…ЁеҚ•иҜҚпјҢдё”иЎҘе…Ёзҡ„еҚ•иҜҚеҸҜд»ҘзӣҙжҺҘзІҳиҙҙеҲ°е…¶д»–ең°ж–№

еҲҶеұҸеҠҹиғҪеҫҲе®һз”Ёе•ҠжңүжңЁжңү

иҮӘеҠЁе®ҢжҲҗ

iTerm2 еҸҜд»ҘиҮӘеҠЁиЎҘйҪҗе‘Ҫд»ӨпјҢиҫ“е…ҘиӢҘе№Іеӯ—з¬ҰпјҢжҢү вҢҳ+; еј№еҮәиҮӘеҠЁиЎҘйҪҗзӘ—еҸЈпјҢеҲ—еҮәеҪ“еүҚеҸҜз”Ёзҡ„е‘Ҫд»ӨгҖӮ

ExposГ© Tabs

вҢҳ+Option+e е…ЁеұҸеұ•зӨәжүҖжңүзҡ„ tabпјҢеҸҜд»Ҙжҗңзҙў

й«ҳдә®еҪ“еүҚйј ж Үзҡ„дҪҚзҪ®

дёҖдёӘж ҮзӯҫйЎөдёӯејҖзҡ„зӘ—еҸЈеӨӘеӨҡпјҢжңүж—¶еҖҷдјҡжүҫдёҚеҲ°еҪ“еүҚзҡ„йј ж ҮпјҢвҢҳ+/ жүҫеҲ°е®ғгҖӮ

й…ҚиүІ

дҪ еҸҜд»ҘиҮӘз”ұе®ҡеҲ¶е–ңж¬ўзҡ„й…ҚиүІпјҢиҝҷйҮҢ ж”¶йӣҶдәҶеӨ§йҮҸ iTerm2 зҡ„дё»йўҳпјҢдҪ еҸҜд»ҘйҖүжӢ©дҪҝз”ЁгҖӮжҲ‘з”Ёзҡ„жҳҜZenburnгҖӮеңЁе…¶ github repo йҮҢдёӢиҪҪеҜ№еә”зҡ„xxx.itermcolorsж–Үд»¶пјҢеҸҢеҮ»е®үиЈ…дҪҝз”ЁгҖӮ

MoomпјҡзӘ—еҸЈи°ғиҠӮзҘһеҷЁ

macOS зі»з»ҹдёҚиғҪеғҸ Windows йӮЈж ·жңҖеӨ§еҢ–жҳҜдёҚжҳҜеҫҲдёҚзҲҪпјҹдёҖиЁҖдёҚеҗҲе°ұе…ЁеұҸпјҒз”Ё Moom жқҘи§ЈеҶіиҝҷдёӘй—®йўҳеҗ§пјҒ

е®үиЈ…еҗҺпјҢе°Ҷйј ж ҮжӮ¬жө®еңЁдҪ жғіи°ғж•ҙзӘ—еҸЈзҡ„е…ЁеұҸжҢүй’®дёҠпјҲе°ұжҳҜйӮЈдёӘз»ҝиүІжҢүй’®пјүпјҢдёӢж–№е°ұдјҡеҮәзҺ°дёҖдәӣжү©еұ•йҖүйЎ№пјҡ

д»Һе·ҰеҲ°еҸідҫқж¬ЎдёәпјҡжңҖеӨ§еҢ–гҖҒе°ҶзӘ—еҸЈе№ій“әеңЁеұҸе№•е·ҰеҚҠиҫ№гҖҒе°ҶзӘ—еҸЈе№ій“әеңЁеұҸе№•еҸіеҚҠиҫ№гҖҒе°ҶзӘ—еҸЈе№ій“әеңЁеұҸе№•дёҠеҚҠиҫ№гҖҒе°ҶзӘ—еҸЈе№ій“әеңЁеұҸе№•дёӢеҚҠиҫ№

иҝҷеҮ дёӘйҖүйЎ№йқһеёёе®һз”ЁпјҢжҜ”еҰӮдҪ жғіжү“ејҖдёҖдёӘзҪ‘йЎөеҗҢж—¶жү“ејҖдёҖдёӘз¬”и®°е·Ҙе…·пјҢиҝҷж—¶дҪ е°ұеҸҜд»ҘзӣҙжҺҘи®©жөҸи§ҲеҷЁеҚ з”Ёе·ҰеҚҠиҫ№пјҢз¬”и®°е·Ҙе…·еҚ з”ЁеҸіеҚҠиҫ№пјҢдёҚйңҖиҰҒиҮӘе·ұжүӢеҠЁжӢ–жӢҪи°ғж•ҙеӨ§е°Ҹе•ҰгҖӮ

PasteпјҡеүӘеҲҮжқҝзҘһеҷЁ

еӨҚеҲ¶зІҳиҙҙжҳҜжҲ‘д»¬ж—Ҙеёёе·ҘдҪңе’ҢејҖеҸ‘дёӯеёёз”Ёзҡ„еҠҹиғҪпјҢPaste дёәжҲ‘д»¬жҸҗдҫӣдәҶеүӘеҲҮжқҝеҺҶеҸІи®°еҪ•зҡ„еҠҹиғҪпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮ cmd+shift+v жқҘжҹҘзңӢи®°еҪ•пјҢйҖҡиҝҮж–№еҗ‘й”®йҖүжӢ©жҲ‘д»¬йңҖиҰҒзҡ„еҶ…е®№еҗҺж•ІеӣһиҪҰе®ҢжҲҗд№ӢеүҚеӨҚеҲ¶еҶ…е®№зҡ„зІҳиҙҙпјҡ

Keyboard Maestroпјҡи®ҫзҪ®еҝ«жҚ·й”®зҘһеҷЁ

жҲ‘ж—Ҙеёёз”Ё HHKB жқҘз Ғеӯ—пјҢиҝҷдёӘй”®зӣҳжңҖеӨ§зҡ„дјҳзӮ№е°ұжҳҜе°Ҹе·§пјҢжңҖеӨ§зҡ„зјәзӮ№д№ҹжҳҜе°Ҹе·§пјҢеҫҲеӨҡй”®жҳҜжІЎжңүзҡ„пјҢжҜ”еҰӮж–№еҗ‘й”®гҖӮ

жҲ‘йҖҡиҝҮ Keyboard Maestro жқҘи®ҫзҪ®дёҖдәӣз»„еҗҲй”®дҪңдёәж–№еҗ‘й”®пјҢеҗҢж—¶и®ҫзҪ®еҸҰеӨ–дёҖдәӣз»„еҗҲй”®дҪңдёә App еҗҜеҠЁзғӯй”®гҖӮ

д»Ӣз»ҚзңӢжҲ‘еҸҰдёҖзҜҮеҚҡе®ўпјҡдҪҝз”Ё-KM-еӨ„зҗҶ-HHKB-ж–№еҗ‘й”®/
